
## Data

Please put the processed multimodal recommendation datasets in this directory.

Expected layout:

```text
data/
  baby/
    baby.inter
    image_feat.npy
    text_feat.npy
  sports/
    sports.inter
    image_feat.npy
    text_feat.npy
  clothing/
    clothing.inter
    image_feat.npy
    text_feat.npy
```

The interaction files and feature files are not included in this repository. If you use public Amazon multimodal recommendation benchmarks, please follow their original license and data usage terms.
